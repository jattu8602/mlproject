## Run Locally

### 1. Install dependencies
```bash
pip3 install -r requirements.txt
```

### 2. Train the model
```bash
python3 src/pipeline/train_pipeline.py
```

### 3. Run the web app
```bash
python3 app.py
```
Open http://localhost:5000 in your browser.

## Deploy to AWS Elastic Beanstalk

### Prerequisites
- AWS account
- AWS CLI + EB CLI installed

### Setup
```bash
pip3 install awscli awsebcli
aws configure          # Enter your AWS Access Key, Secret Key, region
eb init                # Select region, Python platform
eb create mlproject-env   # Create environment (takes ~5 mins)
eb open                # Open the deployed app
```

### Update deployment
```bash
eb deploy
```

### Clean up (to avoid charges)
```bash
eb terminate mlproject-env
```

## Deploy to Azure App Service

### Method 1: Manual via Azure CLI

1. **Create an Azure Web App**
```bash
# Login
az login

# Create resource group
az group create --name mlproject-rg --location eastus

# Create App Service plan (Free tier F1)
az appservice plan create --name mlproject-plan --resource-group mlproject-rg --sku F1 --is-linux

# Create Web App with Python runtime
az webapp create --name <your-app-name> --resource-group mlproject-rg --plan mlproject-plan --runtime "PYTHON:3.9"

# Deploy
az webapp up --name <your-app-name> --resource-group mlproject-rg
```

2. **Configure startup command**
```bash
az webapp config set --resource-group mlproject-rg --name <your-app-name> --startup-file "gunicorn app:application"
```

### Method 2: GitHub Actions (CI/CD)

Create `.github/workflows/deploy-azure.yml`:

```yaml
name: Deploy to Azure Web App

on:
  push:
    branches:
      - main
  workflow_dispatch:

jobs:
  build-and-deploy:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v3

      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.9'

      - name: Install dependencies
        run: pip install -r requirements.txt

      - name: Deploy to Azure
        uses: azure/webapps-deploy@v2
        with:
          app-name: <your-app-name>
          publish-profile: ${{ secrets.AZUREAPPSERVICE_PUBLISHPROFILE }}
```

**Setup Steps:**
1. Go to Azure Portal → App Service → Download **Publish Profile**
2. In GitHub repo → Settings → Secrets → Add `AZUREAPPSERVICE_PUBLISHPROFILE`
3. Push to `main` branch → automatic deployment

### Azure Portal Setup
1. Go to portal.azure.com → Create a Web App
2. Choose: Python 3.9, Linux, Free F1 tier (no cost)
3. In Settings → Configuration:
   - Startup Command: `gunicorn app:application`
4. Deploy via Git or zip deploy

## Dataset

**Student Performance Dataset** (1000 rows, 8 columns)

- **Target:** `math_score` (continuous — regression task)
- **Features:**
  - Categorical: `gender`, `race_ethnicity`, `parental_level_of_education`, `lunch`, `test_preparation_course`
  - Numerical: `reading_score`, `writing_score`

## Models Used

| Model | Hyperparameter Tuning |
|-------|----------------------|
| Linear Regression | — |
| Decision Tree | criterion |
| Random Forest | n_estimators |
| Gradient Boosting | learning_rate, subsample, n_estimators |
| XGBoost | learning_rate, n_estimators |
| CatBoost | depth, learning_rate, iterations |
| AdaBoost | learning_rate, n_estimators |

Best model selected by R² score (minimum threshold: 0.6).
